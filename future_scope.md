## Future Scope

The current face verification system is designed for controlled, close-range identity authentication inside a locomotive cabin.

Possible future enhancements include:

- Multi-frame verification to improve robustness
- Liveness detection (eye blink / head movement)
- Deep learning–based face embeddings for distance-invariant recognition
- Multi-camera setup (front and side views)
- Integration with infrared cameras for low-light conditions
- Centralized driver identity database
- Direct hardware integration with alcohol detection systems

These enhancements can further improve accuracy, security, and scalability for real-world railway deployment.
